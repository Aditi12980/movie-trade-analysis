# 🎬 Movie Rating & Trade Analysis

This project focuses on analyzing movie ratings and box office performance using Excel, SQL, and Power BI to extract meaningful insights and visualize trends.

## 📊 Tools & Technologies
- Excel – Data cleaning and preprocessing  
- SQL – Data querying and analysis  
- Power BI – Interactive dashboard and visualization  

## 📁 Project Structure
- **dataset/** → Raw and cleaned datasets  
- **sql/** → SQL queries and analysis files  
- **dashboard/** → Power BI (.pbix) file  
- **notebook/** → Jupyter notebook (analysis)  
- **screenshots/** → Dashboard preview images  

## 📈 Key Insights
- Identified top-performing movies based on revenue  
- Analyzed genre-wise trends and popularity  
- Explored relationship between ratings and earnings
